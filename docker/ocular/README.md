# Ocular

A fancy, simplistic, beautiful and straight-forward, open-source budgeting tracking app to track your budget across the years.

[Official Documentation](https://simonwep.github.io/ocular)
[GitHub Repository](https://github.com/simonwep/ocular)

## Quick Start

1. (Optional) Copy `.env.example` to `.env` and adjust port if needed:
   ```bash
   cp .env.example .env
   ```

2. Start the service:
   ```bash
   docker compose up -d
   ```

3. Access Ocular at `http://localhost:8080` (or your configured port)

## Features

- Self-hosted budgeting tracker
- Beautiful light and dark theme
- Installable as PWA
- Multi-language and currency support
- Privacy mode
- Keyboard navigation

## Configuration

Data is stored in `./data` directory. The application will initialize on first start.
