# MinIO

S3-compatible object storage server.

## Quick Start

```bash
cd minio
cp .env.example .env  # Optional: customize credentials
docker compose up -d
```

## Configuration

The service can be configured using environment variables. Copy `.env.example` to `.env` and modify as needed:

- `MINIO_ROOT_USER`: MinIO root user (default: minioautumn)
- `MINIO_ROOT_PASSWORD`: MinIO root password (default: minioautumn)

## Access

- **API Port**: 9000
- **Console Port**: 9001 (if enabled)
- **Container name**: `minio-minio-1` (or as generated by Docker Compose)

## Buckets

The service automatically creates a bucket named `revolt-uploads` on startup.

## Usage

Access MinIO using the MinIO Client (mc) or any S3-compatible client:

```bash
docker exec -it <container-name> mc alias set minio http://minio:9000 <MINIO_ROOT_USER> <MINIO_ROOT_PASSWORD>
```
