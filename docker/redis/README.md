# Redis

An in-memory data structure store, used as a database, cache, and message broker.

## Quick Start

```bash
cd redis
docker compose up -d
```

## Configuration

Redis is configured with:
- Automatic persistence: saves data every 60 seconds if at least 1 key changed
- Log level: warning
- Data persistence: stored in Docker volume

## Access

- **Port**: 6379 (default Redis port)
- **Container name**: `redis-redis-1` (or as generated by Docker Compose)

## Usage

Connect to Redis using the Redis CLI:

```bash
docker exec -it <container-name> redis-cli
```
