# MongoDB

A document-oriented NoSQL database.

## Quick Start

```bash
cd mongodb
docker compose up -d
```

## Configuration

MongoDB is configured with a replica set (`rs0`) for high availability. The service runs on the default MongoDB port 27017.

## Access

- **Port**: 27017
- **Container name**: `mongodb-mongo-1` (or as generated by Docker Compose)

## Replica Set

The MongoDB instance is configured with replica set `rs0`. To initialize the replica set, connect to the container and run:

```bash
docker exec -it <container-name> mongosh --eval "rs.initiate()"
```
