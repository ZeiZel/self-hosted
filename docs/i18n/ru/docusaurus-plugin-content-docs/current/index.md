# Введение

Добро пожаловать в документацию Self-hosted Infrastructure!

Этот репозиторий предоставляет полное решение для self-hosted инфраструктуры для разработки и production окружений. Он включает комплексный набор сервисов и инструментов для создания, развертывания и управления приложениями в вашем собственном окружении.

## Что вы найдете здесь

- **Системы контроля версий** - GitLab для управления кодом
- **Облачные хранилища** - Syncthing, ownCloud для синхронизации данных
- **Инструменты дизайна** - Excalidraw для совместной работы над дизайном
- **Коммуникация** - Stoat (Revolt) для командного чата и совместной работы
- **Безопасная аутентификация** - Authentik для SSO между сервисами
- **Системы мониторинга** - Grafana, Prometheus, Loki для наблюдаемости
- **Инструменты разработки** - TeamCity, YouTrack для CI/CD и управления проектами

## Возможности

- **Несколько методов развертывания** - Docker Compose, Kubernetes, Ansible, Terraform
- **Готово к production** - Все сервисы настроены с лучшими практиками
- **Комплексный мониторинг** - Встроенная наблюдаемость и логирование
- **Удобно для разработчиков** - Полный CI/CD pipeline и инструменты разработки
