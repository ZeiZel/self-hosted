# Services Available Here

- [Glance](https://github.com/zurdi15/glance) - modern dashboard with all services (replacement for Dashy)
- Productivity
  - [Notesnook](https://notesnook.com/) - convenient open-source product for real-time note-taking
  - [Excalidraw](https://github.com/alswl/excalidraw-collaboration) - convenient replacement for FigJam
  - [YouTrack](https://www.jetbrains.com/youtrack/)
- Design
  - [Penpot](https://penpot.app/) - Figma alternative with open source, and self-hosted!
- Passwords
  - [Vaultwarden](https://github.com/dani-garcia/vaultwarden)
- [Remnawave](https://remna.st/) - proxy management
- Communications
  - [Stoat (Revolt)](https://github.com/stoatchat/self-hosted) - Discord alternative with open source and better features!
- Mail Server
  - [Stalwart](https://stalw.art/) - mail server that includes everything
- Code Management
  - [Gitlab](https://about.gitlab.com/install/) - VCS for our codebases
  - [TeamCity](https://www.jetbrains.com/ru-ru/teamcity/) - CI/CD organizer
  - [JB Hub](https://www.jetbrains.com/hub/) - manager for connecting YouTrack, Teamcity and other related services
- Shared Storage
  - [Harbor](https://goharbor.io/) - container registry
  - [Sonatype Nexus](https://help.sonatype.com/en/installing-iq-server.html)
- Remote Data Storage
  - [Syncthing](https://syncthing.net/)
  - [ownCloud](https://owncloud.com/)
- Authentication
  - [Authintik](https://goauthentik.io/) - great solution that supports SAML2 we need
- Databases
  - [PostgreSQL](https://www.postgresql.org/)
  - [MiniO](https://www.min.io/)
  - [ValKey](https://github.com/valkey-io/valkey)
  - [MongoDB](https://www.mongodb.com/)
  - [Supabase](https://github.com/supabase/supabase)

Helper Tools:

- [Monitoring](https://habr.com/ru/articles/766102/)
  - [Devtron](https://devtron.ai/) - Kubernetes dashboard and CI/CD platform
  - [Bytebase](https://www.bytebase.com/) - database schema management
  - [Prometheus](https://prometheus.io/docs/prometheus/latest/installation/)
  - [Loki](https://grafana.com/docs/loki/latest/setup/install/helm/)
  - [Grafana](https://grafana.com/docs/grafana/latest/setup-grafana/installation/kubernetes/)
- Secrets
  - [Vault](https://developer.hashicorp.com/vault)
- Service Delivery
  - [Consul](https://developer.hashicorp.com/consul) (+Fabio, Register and Prometheus exporter)
- Load Balancer
  - [Traefik](https://github.com/traefik/traefik)
