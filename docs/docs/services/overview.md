# Services Overview

This infrastructure includes a comprehensive set of self-hosted services organized by category.

## Productivity

- **[Notesnook](https://notesnook.com/)** - Open-source note-taking application with real-time sync
- **[Excalidraw](https://github.com/alswl/excalidraw-collaboration)** - Collaborative whiteboard tool (Figma alternative)
- **[YouTrack](https://www.jetbrains.com/youtrack/)** - Issue tracking and project management

## Communication

- **[Stoat (Revolt)](https://github.com/stoatchat/self-hosted)** - Open-source Discord alternative with self-hosting support

## Storage

- **[Syncthing](https://syncthing.net/)** - Continuous file synchronization
- **[Nextcloud](https://nextcloud.com/)** - File sharing and collaboration platform
- **[MinIO](https://www.min.io/)** - S3-compatible object storage

## Development

- **[GitLab](https://about.gitlab.com/install/)** - Complete DevOps platform with Git repository, CI/CD, and more
- **[TeamCity](https://www.jetbrains.com/teamcity/)** - CI/CD server and build management
- **[JetBrains Hub](https://www.jetbrains.com/hub/)** - Unified authentication and user management for JetBrains tools

## Infrastructure

- **[Traefik](https://github.com/traefik/traefik)** - Reverse proxy and load balancer
- **[Consul](https://developer.hashicorp.com/consul)** - Service discovery and configuration management
- **[Vault](https://developer.hashicorp.com/vault)** - Secrets management
- **[Authentik](https://goauthentik.io/)** - SSO and identity provider

## Monitoring & Logging

- **[Grafana](https://grafana.com/)** - Analytics and monitoring dashboards
- **[Prometheus](https://prometheus.io/)** - Metrics collection and alerting
- **[Loki](https://grafana.com/docs/loki/)** - Log aggregation system
- **[ELK Stack](https://www.elastic.co/elastic-stack)** - Elasticsearch, Logstash, Kibana for log management

## Databases

- **[PostgreSQL](https://www.postgresql.org/)** - Advanced open-source relational database
- **[MongoDB](https://www.mongodb.com/)** - NoSQL document database
- **[ValKey](https://github.com/valkey-io/valkey)** - In-memory data structure store
- **[MinIO](https://www.min.io/)** - Object storage (also listed under Storage)

## Additional Tools

- **[Glance](https://github.com/zurdi15/glance)** - Modern dashboard for all services
- **[Vaultwarden](https://github.com/dani-garcia/vaultwarden)** - Password manager (Bitwarden-compatible)
- **[Harbor](https://goharbor.io/)** - Container registry
- **[Bytebase](https://www.bytebase.com/)** - Database schema management
- **[Devtron](https://devtron.ai/)** - Kubernetes dashboard and CI/CD platform

## Service Categories

- [Productivity Services](./productivity) - Notes, drawing, and project management
- [Communication Services](./communication) - Chat and messaging
- [Storage Services](./storage) - File sync and object storage
- [Development Services](./development) - CI/CD and code management
- [Infrastructure Services](./infrastructure) - Service discovery, secrets, and routing
- [Monitoring Services](./monitoring) - Observability and logging
